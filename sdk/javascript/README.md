# `@fide-work/context-protocol`

JavaScript/TypeScript SDK for building and validating Fide Context Protocol statements.

- SDK docs: [fide.work/docs/context-protocol/sdk/javascript](https://fide.work/docs/context-protocol/sdk/javascript)
- npm: [@fide-work/context-protocol](https://www.npmjs.com/package/@fide-work/context-protocol)
- Repository: [github.com/fide/context-protocol](https://github.com/fide/context-protocol)

## Status

This package is in alpha.

- Breaking changes are allowed during alpha.
- Consumers should pin exact versions.

## Install

```bash
pnpm add @fide-work/context-protocol
```

## Development

Useful local commands:

- `pnpm run build`
- `pnpm test`
